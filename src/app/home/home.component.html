Learn To Code!

This is a prototype of the Learn To Code app which is under development.

There is now interaction (albeit shakey) between client and server.

Additionally, the question/answer framework which is currently in use does not require code submission, but is instead based on simple one word answers.

You will need to login as a teacher to create courses, sequences and questions.

You will need to login as a student to answer questions.

Example login details are given on the login page.
